About

This simulation website was created for PHYS-495 (senior final project) at the University of Southern California by Emilia Doda, a physics and computer science major. This project intends to act as a visual educational supplement for teaching quantum tunneling and gaussian wave packets.


Languages and technologies used:

  • Python: Used for creating the simulations.
  • Node.js: A JavaScript runtime environment used to create the web application.
  • Vercel: A cloud platform used for deploying static web projects.
  • Mathjax: A JavaScript display engine for mathematics.
  • Flask: A Python web framework that allows users to run Python in various environments.
  • Plotly: An open-source Python visualization library/module for creating interactive graphs and animations.
  • Pyodide: A Python distribution for the browser and Node.js based on WebAssembly. Allows Python to run in web browser.
  • NumPy: A python library used for general numerical calculations.
  • SciPy: A python library used for specific scientific calculations. I used this specifically for linear algebra.
