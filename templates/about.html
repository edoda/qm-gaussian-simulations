About

This website, developed for PHYS-495 (senior final project) at the University of Southern California, showcases interactive simulations created by Emilia Doda, a physics and computer science student. This website acts as an educational supplement to visually enrich the learning of undergraduate-level quantum mechanics, with a focus on illustrating the principles of quantum tunneling and Gaussian wave packets.

Languages and technologies used:

  • Python: Used for creating the simulations.
  • Vercel: A cloud platform used for deploying this static web project.
  • Mathjax: A JavaScript display engine for mathematics.
  • Flask: A Python web framework that allows users to run Python in various environments.
  • Plotly: An open-source Python visualization library/module for creating interactive graphs and animations.
  • Pyodide: A Python distribution for the browser and Node.js based on WebAssembly. Allows Python to run in web browser.
  • NumPy: A python library used for general numerical calculations.
  • SciPy: A python library used for specific scientific calculations.
